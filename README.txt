nwc2xml is a conversion program that reads nwc(Noteworthy Composer:http://www.noteworthysoftware.com/) file
and generate music xml(http://www.recordare.com/xml.html) output.
Version 1.5 (March 2012) supports reading nwc v2.1 and output musix xml 1.1 compatible.

NWC2XML is distributed under the terms of the GNU GPL v2.
This includes everything in the source code distribution.
